Jain-Faces

  • This is an extension library to Prime-Faces to support some messing components from both Prime-Faces and Prime-Faces Extension.
  • All the components available in this library are Mobile friendly and created using Responsive and Adaptive CSS.
  • This component library Includes following Components
    • JN-Layout
    • JN-Grid
    • JN-Output
    • JN-Group
  • This library Includes
    • Couple of most useful common methods needs while JSF application development
  • Common Convertor
    • This can be used in any JSF application or object to convert them back and forth
    • This needs a common interface JNIIdentificable to be impmeneted by any object
